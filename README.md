# AsciiCipher

This program takes ASCII values, converts them into text, and then applies a Caesar cipher (ROT13) to the decoded message.

ASCII → TEXT → CIPHER

## Example

Input:
72 69 76 76 79 32 87 79 82 76 68

Output:
DECODED TEXT:
HELLO WORLD

CAESAR CIPHER (SHIFT 13):
URYYB JBEYQ

## Another Example

Input:
73 32 76 79 86 69 32 67 79 68 73 78 71

Output:
DECODED TEXT:
I LOVE CODING

CAESAR CIPHER (SHIFT 13):
V YBIR PBQVAT

## How to run

1. Compile: javac messageDecoder.java  
2. Run: java messageDecoder  
3. Enter the ASCII values when asked.

## What you learn

- ASCII decoding
- String handling in Java
- Basics of cryptography (Caesar / ROT13)
- Multi-step message processing

> Note: ROT13 is only for learning — not real security.
